## How to run the project locally

1. Download or clone this repository
    - to clone the repo: git clone https://github.com/LeanneJamesLlena/frontend-course-project.git
2. Open the folder in an editor(for example VS Code)
3. Navigate to the homepage: "frontend/pages/index.html"
4. Open index.html in a browser
    - Easiest: Right click -> "Open with Live Server"(if you use the Live Server extension in VS Code), or
    - Double-click index.html in your file manager to open it in the browser.
5. Use the Navigation links(Home, Services, About, Contact) to move between pages.

Thats it, no terminal commands or build steps needed.
